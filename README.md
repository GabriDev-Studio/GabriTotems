# GabriTotems

Плагин на тотемы от t.me/GabriStudio
Поддержка: 1.16.5 java 17+

Необходимые плагины: PlacholderAPI(для плайсхолдеров)
команды: 
	Основаная: /totem 
	Перезагрузка: /totem reload 
	Выдача тотема: /totem give <игрок> <названние тотма из config.yml> 
	Проверка Обновлений: /totem update

Если что-то не работает пишите в телеграм: t.me/DoXeRoMaN (через переходник будет мой юз) - ПОЖАЛУЙСТА ПИШИ, Я ТЕБЯ ПРОШУ
